# 📊 Supply Chain & Inventory Analysis Dashboard

## Overview

This project presents an interactive **Supply Chain & Inventory Analysis Dashboard** developed using **Power BI**. The dashboard provides comprehensive insights into inventory performance, warehouse utilization, transportation costs, sales trends, and order fulfillment metrics to support data-driven decision-making.

The objective of this project is to help businesses optimize inventory management, improve operational efficiency, and monitor key supply chain KPIs through intuitive visualizations.

---

## 🎯 Business Objectives

* Monitor warehouse utilization and storage efficiency.
* Analyze inventory levels across product categories and regions.
* Track transportation costs and identify cost-intensive regions.
* Evaluate sales performance and demand trends.
* Measure inventory turnover and Days Sales in Inventory (DSI).
* Monitor order fulfillment status and backorders.

---

## 📈 Key Performance Indicators (KPIs)

### Warehouse Performance

* Warehouse Utilization (%)

### Inventory Management

* Inventory Level by Category
* Inventory Level by Region
* Inventory Turnover Ratio
* Days Sales in Inventory (DSI)

### Logistics & Transportation

* Transportation Cost by Region
* Transportation Cost by Product Category

### Sales Performance

* Units Sold by Year
* Category-wise Sales Analysis

### Order Management

* Backorder Count
* Order Status Analysis (Fulfilled, Pending, Cancelled)

## 📊 Dashboard Features

* Interactive slicers for Region and Category filtering.
* Dynamic KPI cards for real-time performance tracking.
* Warehouse utilization gauge chart.
* Transportation cost analysis across regions.
* Inventory distribution analysis by category.
* Sales trend visualization.
* Order fulfillment and backorder monitoring.
* Regional inventory comparison.

## 📌 Insights Generated

* Identified categories with the highest inventory holding levels.
* Evaluated warehouse space utilization efficiency.
* Compared transportation costs across different regions.
* Monitored yearly sales growth and demand patterns.

* Measured inventory efficiency using turnover ratio and DSI metrics.

## 🚀 Business Impact

This dashboard enables supply chain managers and business stakeholders to:
